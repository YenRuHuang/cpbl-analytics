wRC+ 公式說明

這些指標的計算邏輯與限制

wOBA — 加權上壘率

wOBA 根據每種進攻事件（BB、HBP、1B、2B、3B、HR）對得分的貢獻程度賦予不同權重， 比 OBP 更準確地反映打者的進攻價值。權重來自 linear weights，代表每種事件平均帶來的額外得分期望值。

wOBA = (0.690×BB + 0.722×HBP + 0.888×1B + 1.22×2B + 1.56×3B + 1.95×HR) ÷ (AB + BB - IBB + SF + HBP)

* 本系統實際使用之權重（來自 JSON meta）。FanGraphs 2024 MLB 標準值為 1.271×2B、1.616×3B、2.101×HR，因 CPBL 環境調整後採用較保守估計。

wRC+ — 加權得分創造指數

wRC+ 將打者的 wOBA 轉換為「相對聯盟平均的進攻指數」，100 為聯盟平均。 110 代表比聯盟平均進攻好 10%，85 代表差 15%。因為已標準化，跨季、跨球場都可比較。

wRC+ = [ (wOBA - lgwOBA) / wOBAscale + lgR/PA ] ÷ lgR/PA × 100

* lgwOBA、wOBAscale、lgR/PA 從當季數據計算

本實作的限制

  • · MLB linear weights：使用 FanGraphs 2024 MLB 權重值。CPBL 的各事件得分期望值與 MLB 可能有差異，理論上需用 CPBL 自身數據重新計算。
  • ✓ 含 Park Factor 調整：使用 team-based Park Factor（主場 RPG ÷ 客場 RPG），校正球場對得分環境的影響。樂天桃園 PF=1.12（打者球場），大巨蛋 PF=0.86（投手球場）。
  • · IBB 未區分：資料來源無法完全區分故意四壞（IBB）與一般四壞（BB），計算時一律歸入 BB。
  • · 樣本量：CPBL 每季約 120 場，需至少 100 打席（PA ≥ 100）才納入排行，排除小樣本雜訊。
